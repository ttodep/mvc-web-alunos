# mvc-web-alunos

## MVC: Model View Controller
  A ideia principal é separar o código em 3 módulos principais
  ### 1. Model:
    É o módulo que tem por objetivo criar estruturas de 
    dados que serão utilizadas pelos outros módulos. 
    Ele é a base para os outros módulos operarem.
  ### 2. View:
    Esse é simples, comunicação entre código e usuário.
    Ele coloca coisas na tela e captura inputs do usuário.
  ### 3. Controller: 
    Esse é onde haverá a maior parte dos métodos, onde ocorrerá 
    tratamento de excessões e fará as principais funções
    que o código deve executar
